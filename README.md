# SecureBank Smart Customer Support Chat System

This project is a Smart Customer Support Chat System for SecureBank.

## Main features
- RAG using SecureBank official documents
- LlamaIndex for document ingestion and indexing
- LangChain for integrations
- LangGraph for orchestration
- Qdrant for vector storage and retrieval
- DeepSeek API as the LLM
- Customer memory and conversation history
